# Silly Timer

A simple and professional PWA Timer application with minimal design.

## Features

- **Time Input Page**: Set hours, minutes, and seconds with +/- buttons (increments of 1 or 5)
- **Timer Page**: Countdown with circular progress indicator
- **Controls**: Pause/Resume and Return to Home functionality
- **PWA Support**: Works offline, installable on mobile devices
- **Design**: Clean white background with grey and blue accent colors

## Installation

### As a Web App
Simply open `index.html` in a web browser or serve it with any HTTP server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`

### As a PWA
1. Open the app in a mobile browser (Chrome, Safari, etc.)
2. Use "Add to Home Screen" option to install
